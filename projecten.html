System engineering & Analysis

We kregen een project vanuit school voorgeschoteld om de temperatuur te regelen in verschillende lokalen. De controle moest gebeuren op automatische basis, met vooraf ingestelde waarden. Deze waarden moeten gewijzigd kunnen worden via een website. Samen met een goep zijn we dan begonnen aan het project. Planning en teamwerk waren zeer belangrijk en ook de werkpunten.

We waren begonnen met het project op te delen in een aantal kleinere projectjes. Dit zorgde ervoor dat iedereen kon werken aan een klein deeltje van het project. Een airco werd bijvoorbeeld gesimuleerd met een DC-motor, een zonnenwering met een stappenmotor met een doek. Temperatuursmetingen werden met temperatuurssensoren gedaan maar later gesimuleerd met potentiometers om makkelijker testen uit te voeren, hetzelfde geld voor de andere sensoren zoals bijvoorbeeld lichtsensoren.

Tijdens dit project heb ik me bezig gehouden met het verzamelen en opslaan van data die verzonden werd van de andere raspberry pi's en de correcte functies te verzenden zodat de andere raspberry pi's wisten welke actie ze moesten volbrengen.

Project IoT

We kregen voor het vak project IoT een aantal projecten voorgesteld waaruit we de keuze kregen om deze uit te werken. Mijn groep heeft beslist om een project waar de groei van pompoenen wordt gemonitord. Ook het automatische toevoegen van water en voedingsstoffen zou mogelijk gemaakt worden in dit project. Het op afstand beschikbaar maken zou via een website mogelijk gemaakt worden.

Door de coronacrisis is het realiseren van dit project zeker niet makkelijk. Maar de het op afstand werken zorgt weer voor een hele nieuwe aanpak voor het uitwerken van het project. Maar de samenwerking en het uitwerken van het project voorloopt zeer goed. Comunicatie is belangrijk in elk project en als deze goed is kan het project goed blijven doorgaan hoe moeilijk de omstandigheden het ook maken al is het wel met enkele aanpassingen en andere verwachtingen ten opzichte van elkaar.

Ik hou me vooral bezig met het maken van de software die de sturing van de waterpompen en de waardes van sensoren binnenhaalt. Ook heb ik gewerkt aan de database zodat deze data gevisualiseerd kon worden op een website.
